# 🛒 Amazon Clone (Frontend)

A simple **Amazon homepage clone** built using **HTML and CSS**.
This project focuses on replicating the UI design and layout of Amazon’s website to improve frontend development skills.

---

## 🚀 Features

* Responsive navigation bar
* Search bar UI
* Hero section with banner image
* Product grid layout
* Footer section
* Clean and structured layout using CSS
* Hover effects for interactive UI elements

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout design

---

## 📂 Project Structure

```
Amazon-Clone/
│── index.html
│── style.css
│── images/
```

---

## 🎯 Purpose of the Project

This project was built to:

* Practice HTML structure and semantic tags
* Improve CSS styling and layout skills
* Understand how real-world e-commerce websites are structured
* Strengthen frontend fundamentals
